# Scorecast - Generate your Match Score Stories from the Sideline

Scorecast lets you generate game score stories directly from the sideline.

Perfect for coaches, teams, media managers, or content creators looking for a simple, fast, and easy way to share match score updates in real time.

No need to wait until the end of the game to create engaging score stories. With Scorecast, you can create and share score updates as the game unfolds.

## Features
- **Real-time Score Updates**: Instantly create and share score stories as the game unfolds.
- **Customizable Colors**: Tailor the appearance of your score stories to match your team's branding (font, colors).
- **Easy-to-Use Interface**: User-friendly design makes it simple to create and share stories quickly.
